# Coa
A repo having all the experiments done in the 4rth semester in the subject - " computer organisation and architecture"


This repository contains all COA lab experiment files in PDF format. Each file includes detailed explanation with aim, theory, implementation and output.


Contents of the repository:- 

1. Digital Adders (Half Adder, Full Adder, 4-bit Adder)
These files explain the design and working of basic arithmetic circuits. It includes implementation using logic gates and NAND-only realization, along with ripple carry behavior in multi-bit addition.

2. Encoder and Multiplexer
Covers the design and functionality of an 8:3 encoder and an 8:1 multiplexer. The experiments demonstrate data selection, encoding techniques, and Boolean logic expressions.

3. 4-bit ripple Adder
To design the Ripple Carry Adder that is a combinational circuit used to add multi-bit binary numbers.

4. GDB Debugging Experiment
Focuses on understanding program execution using GNU Debugger. It includes use of breakpoints, step execution, and tracking variable changes during runtime.

5. Addressing Modes using GDB
Explains different instruction addressing modes such as immediate, register, indirect, indexed, and stack addressing. Assembly instructions and register values are analyzed to understand low-level execution.

6. 4-bit Common Bus System
Describes how multiple registers share a common communication path. It explains data transfer using multiplexers and selection lines, showing efficient hardware organization.

Program Outputs and Analysis :-
Each experiment includes actual outputs and observations obtained during execution. This helps in verifying correctness and understanding practical behavior.

Tools and Environment :-
Logisim used for digital circuit simulation
C/C++ programs compiled with debugging flags
GNU Debugger (GDB) for execution analysis
Linux environment for development and testing
